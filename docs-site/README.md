# Physical AI & Humanoid Robotics Textbook

Welcome to the Physical AI & Humanoid Robotics textbook - an interactive, AI-native learning platform designed to bridge the gap between digital intelligence and physical embodiment.

## Overview

This textbook is structured around 5 core modules that progressively build your expertise in Physical AI and embodied intelligence:

1. **ROS 2 (Robotic Nervous System)** - Establish the communication backbone for robotic systems
2. **Gazebo & Unity (Digital Twin)** - Create simulation environments for testing and development
3. **NVIDIA Isaac (AI Robot Brain)** - Implement intelligent decision-making and perception
4. **Vision-Language-Action (VLA)** - Integrate multimodal AI for real-world interaction
5. **Capstone: Autonomous Humanoid** - Apply all concepts in a comprehensive final project

## Features

- Interactive textbook content with hands-on labs
- Embedded RAG chatbot for instant Q&A
- Personalized learning paths based on your background
- Urdu translation support for global accessibility
- 13-week structured curriculum from beginner to intermediate

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Visit http://localhost:3000 to view the textbook
